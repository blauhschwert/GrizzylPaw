# GrizzlyPaw — 2D Top-Down Adventure (Godot)

## Overview

**GrizzlyPaw** is a 2D top-down action-adventure game built with the Godot Engine.
You play as a lone bear exploring a mysterious dungeon filled with dangers, secrets, and ancient remnants of a forgotten world.

The game focuses on exploration, simple combat mechanics, and atmosphere, combining classic dungeon-crawler elements with a primal theme.

---

## 🎮 Features

* **Top-down 2D gameplay**
* **Player-controlled bear character**
* **Dungeon exploration with multiple rooms**
* **Basic combat system (melee attacks)**
* **Enemies with simple AI behavior**
* **Scene transitions between dungeon areas**

---

## Gameplay

You control a bear trapped (or perhaps drawn) into an ancient dungeon.
As you explore deeper, you’ll encounter hostile creatures, narrow corridors, and hidden paths.

The goal is simple:

* Survive
* Explore
* Find a way through the dungeon

There is no hand-holding — the environment tells the story.
